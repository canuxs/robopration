# Robopration 🛡️

**Robopration** is a secure Command Line Interface (CLI) tool designed to provide a protective barrier while browsing the web. It launches an isolated Incognito browser with customizable security toggles using Puppeteer.

## 🚀 Key Features
- **Incognito by Default:** No history, no cache, completely fresh sessions.
- **Cookie Barrier:** Blocks cookies from being sent to the server when disabled.
- **Script Barrier:** Disable JavaScript to prevent tracking and malicious scripts.
- **English-First CLI:** Simple and intuitive commands for developers.

## 📦 Installation
Install globally via npm to use the `robopration` command anywhere:
```bash
npm install -g robopration
